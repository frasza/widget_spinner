# widget_spinner

Widget Spinner animation!

Inspired by fidget spinners.

## Usage / Example

```dart
WidgetSpinner(
    // How many turns will the animation go for
    turns: 10,
    // How long will the animation last
    duration: 10,
    // Widget that spins
    child: Container(
        width: 200,
        height: 200,
        color: Colors.red,
    ),
)
```
